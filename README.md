# Atividade de POO em C++

Este projeto contém implementações básicas de conceitos de polimorfismo em C++.

## Estrutura

- **Parte 1** – Templates (função maiorValor)
- **Parte 2** – Sobrecarga de funções (somar)
- **Parte 3** – Sobrecarga de operadores (classe Ponto)
- **Parte 4** – Template de classe (Caixa)
